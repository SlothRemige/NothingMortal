說到廚神必備，怎能少得了大名鼎鼎、萬能的蠔油
下麵簡單帶一下蠔油的廣泛應用
**火鍋（湯底 + 蘸料）**
餐桌上刷燙的火鍋，無論湯底是用肉類、蔬菜或是海鮮熬煮，都是發揮蠔油鮮美味的最佳方式。蠔油更是火鍋蘸料的靈魂所在，是讓口感更加豐富的關鍵！
**醃製**
蠔油獨特的鮮甜口感，能提升食材原有的鮮味; 在醃製時還可以使得肉類更鮮嫩多汁，是醃製的必備醬料。
**炒菜**
作為炒菜的經典必備，蠔油能夠很好地包裹食材，在翻炒中鎖住食材鮮味、增添美味。
**燉煮**
燉煮菜式中添加蠔油，不僅提升食材的鮮味讓味道更濃郁，而且能夠讓肉質鮮美滑嫩有光澤。
**沙拉醬料，各種調味，拌菜或其他**
蠔油還適用於任何料理及香料搭配，無論製作風味濃郁的沙拉醬，或混醬淋汁於越南米紙卷，或拌製蠔油撈麵，蠔油都起到提鮮提味的作用。
